# Apartment Assist

A Full Stack Apartment Management Platform built using the MERN Stack.

## Features

### Tenant
- Tenant Registration and Login
- View Apartment Events
- Raise Complaints
- Submit Suggestions
- View Reminders
- Access Emergency Contacts
- View Personal Profile

### Secretary
- Secretary Registration and Login
- Add and Manage Events
- View and Update Complaints
- View Resident Suggestions
- Add and Manage Reminders
- Manage Emergency Contacts
- View Secretary Profile

## Technologies Used

### Frontend
- React.js
- React Router
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

## Project Structure

Apartment Assist
├── frontend

└── README.md

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
